Política de Privacidade para VarnishCalc

Data de vigência: [3. Junho de 2024]

1. Introdução

Bem-vindo ao VarnishCalc. Valorizamos a sua privacidade e estamos comprometidos em proteger os seus dados pessoais. Esta Política de Privacidade informa-o sobre as nossas políticas relacionadas com a recolha, uso e divulgação de dados pessoais quando utiliza a nossa aplicação.

2. Recolha de Dados

Não recolhemos quaisquer dados pessoais dos utilizadores. O VarnishCalc não solicita, rastreia ou armazena qualquer informação pessoal ou dados de utilização.

3. Utilização de Dados

Como não recolhemos quaisquer dados, não utilizamos os seus dados de nenhuma forma. A aplicação funciona sem necessitar de acesso às suas informações pessoais.

4. Proteção de Dados

A privacidade e a segurança dos seus dados são importantes para nós. Embora não recolhamos nenhuma informação pessoal, implementámos medidas para garantir que qualquer interação de dados dentro da aplicação permaneça segura.

5. Alterações a Esta Política de Privacidade

Poderemos atualizar a nossa Política de Privacidade periodicamente. Notificá-lo-emos sobre quaisquer alterações, publicando a nova Política de Privacidade nesta página. As alterações a esta Política de Privacidade são eficazes quando publicadas nesta página.
